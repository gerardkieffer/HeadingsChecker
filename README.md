# HeadingsChecker

> **Study fork** — This is a personal study fork of [HeadingsMap](https://addons.mozilla.org/en-US/firefox/addon/headingsmap/) by Jorge Rumoroso. It is not intended for release or redistribution.

HeadingsMap is the original extension, created and maintained by Jorge Rumoroso. All rights reserved by the original author.

## Purpose

This fork was created to study the extension's architecture and experiment with a Safari port and visual rebrand. It is not published or distributed.

## Build

Requires Xcode on macOS. No external build tools are needed.

1. Open `HeadingsMap.xcodeproj` in Xcode
2. Select the `HeadingsMap` scheme and a macOS destination
3. Build: `Cmd+B` — or `xcodebuild -scheme HeadingsMap -destination 'platform=macOS'`
4. Run: `Cmd+R`, then enable the extension in Safari's Extension preferences
