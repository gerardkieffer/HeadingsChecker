O tym forku

HeadingsChecker to osobisty fork do nauki oparty na HeadingsMap autorstwa Jorge Rumoroso. Nie jest to oprogramowanie open source i nie jest przeznaczone do redystrybucji.

Oryginalne rozszerzenie jest dostępne w Firefox Add-ons i Chrome Web Store. Wszelkie prawa zastrzeżone przez oryginalnego autora.
