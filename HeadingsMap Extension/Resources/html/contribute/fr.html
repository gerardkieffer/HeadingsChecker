À propos de ce fork

HeadingsChecker est un fork d'étude personnel de HeadingsMap par Jorge Rumoroso. Ce n'est pas un logiciel libre et il n'est pas destiné à être redistribué.

L'extension originale est disponible sur Firefox Add-ons et le Chrome Web Store. Tous droits réservés à l'auteur original.
